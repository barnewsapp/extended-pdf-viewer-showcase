By default, the PDF file is displayed with 100% zoom. The PDF viewer uses 100% of the width of its container. If the container is wider than the PDF file, the PDF viewer is centered horizontally.


Activating the text layer slows down rendering slightly, but enables the user to mark text. Note that this doesn't work with every PDF file: many PDF files are actually simply a collection of images (even though the may display text), so it's not possible to mark text.
