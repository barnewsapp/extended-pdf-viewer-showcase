By default, moving the mouse wheel makes the PDF file scroll. If you keep the CTRL or CMD key pressed while rotating the mouse wheel, you change the zoom level instead.

If you don't like the default, you can switch the default action.

[wheelAction]="'zoom'"
[wheelAction]="'scroll'" (default)
